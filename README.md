# AE-CNN Ensemble-Based Intrusion Detection System

##  Overview
An AI-based intrusion detection system for multi-class network traffic classification using Autoencoders and CNN ensemble techniques.

##  Features
- Reduced 10,000+ features using Autoencoders  
- CNN + Ensemble Voting mechanism  
- Achieved 96–98% accuracy  
- Reduced false positives significantly  
- Handled class imbalance using SMOTE  

##  Tech Stack
- Python  
- TensorFlow / Keras  
- Scikit-learn  
- Pandas  

##  Outcome
Improved efficiency and reliability of intrusion detection systems using advanced AI techniques.
